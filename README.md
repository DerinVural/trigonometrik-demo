# Trigonometrik İşlemler

Bu proje, temel trigonometrik hesaplamaları yapan bir Python scripti içerir.

## Özellikler

- **Sinüs Hesaplama:** Verilen açının sinüs değerini hesaplar.
- **Kosinüs Hesaplama:** Verilen açının kosinüs değerini hesaplar.
- **Tanjant Hesaplama:** Verilen açının tanjant değerini hesaplar.
- **Esnek Giriş:** Açılar derece veya radyan cinsinden verilebilir (Varsayılan: Radyan, ancak script içi örneklerde derece kullanımı gösterilmiştir).

## Dosyalar

- `trigonometri.py`: Ana uygulama dosyası.

## Kullanım

Scripti çalıştırmak için terminalde şu komutu kullanın:

```bash
python3 trigonometri.py
```

### Örnek Çıktı

```text
Trigonometrik İşlemler Scripti
------------------------------
Açı: 30 derece
  Sinüs:   0.5000
  Kosinüs: 0.8660
  Tanjant: 0.5774
...
```
